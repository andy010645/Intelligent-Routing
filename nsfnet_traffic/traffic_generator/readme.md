## Traffic Generator

# Install

需要用到TMGEN套件 https://github.com/progwriter/TMgen  

此套件需要安裝在python2下，先執行pip install --upgrade setuptools  

接著按照github教學安裝tmgen

python package requirement
* numpy
* cython
* matplotlib
* statistics

# How to use

先執行generate_tms.py，再執行iperf3_scripts.py  
就會成功產生TM資料夾
